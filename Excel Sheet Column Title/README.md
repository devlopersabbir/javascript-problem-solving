# Excel Sheet Column Title

User can give any number as argument in our function. And it should be return
corresponding column title as it appears in an Excel sheet.

For example:

| Letter | ColumnNumber |
| ------ | ------------ |
| A      | 1            |
| B      | 2            |
| C      | 3            |
| Z      | 26           |
| AA     | 27           |
| AB     | 28           |
